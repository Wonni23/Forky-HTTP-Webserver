Cookie/Session Demo (Light Version)

Purpose: This demonstrates HTTP cookies without server-side session storage. All data is stored in browser cookies only.

Test Cookie Operations:

View Current Cookies Set Cookies Delete Cookies

How It Works:

  • CGI script sets Set-Cookie header in HTTP response
  • Browser automatically stores cookies
  • Browser sends cookies back via Cookie header in subsequent requests
  • CGI reads cookies from HTTP_COOKIE environment variable
  • No server-side storage - all data lives in cookies

Technical Details:

  • Language: Python CGI script
  • Cookie Attributes: Path, Max-Age, HttpOnly
  • C++ Changes: Only HttpResponse (addCookie method)
  • Difference from Full Version: No SessionManager, no user authentication, no server-side state
